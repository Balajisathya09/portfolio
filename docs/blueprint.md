# **App Name**: Fluxion Portfolio

## Core Features:

- Narrative Scroll Engine: An high-performance UI implementation that reveals text word-by-word as the user scrolls, creating a rhythmic storytelling effect.
- Cinematic Page Transitions: Fluid, 'butter-smooth' entrance and exit animations between Home, Projects, and About sections using framer-motion.
- Project Lore AI Tool: A feature that uses a generative AI tool to automatically write compelling technical project descriptions based on provided code snippets or GitHub README links.
- Interactive Bento-Grid Projects: A sophisticated gallery of works presented in an interactive grid layout inspired by high-end consumer tech aesthetics.
- Kinetic Typography Logo: A custom-coded SVG animation that reacts to mouse proximity, emphasizing a high-end designer brand identity.
- Micro-Interaction Contact Portal: A contact page that utilizes subtle hover-triggered sound effects and physicalized input fields for a tactile feel.

## Style Guidelines:

- Primary Color: Vibrant Solar Yellow (#FFD700) used for high-impact emphasis and text reveals.
- Background: Deep Charcoal (#141311) for a sophisticated, luxury feel, accented with Pure Black (#000000) in section boundaries.
- Accent: Burnt Orange (#E67E22), chosen as an analogous companion to Yellow to create depth in highlights and icons.
- Font pairing: 'Space Grotesk' for technical, bold headlines; 'Inter' for neutral, highly legible body text to ensure an objective feel.
- Custom stroke icons with thin 1px weight, featuring subtle yellow glow effects on hover.
- Minimalist wide-margin grid layout that focuses heavily on negative space and massive typographic scale.
- Low-friction inertial scrolling with easing functions inspired by the Apple hardware feel (stiffness: 100, damping: 20).
